How to use?

1. Open unity.
2. Build the project to iOS project.
3. Open the project in Xcode.
5. Add "libiOSDeviceInfo.a" to the project.
6. Add "iOSDeviceInfo.h" file to the project.
7. Add "iOSDeviceInfoInterface.mm" file to the project.
8. Then build it.
9. You'll see the app is running well in your device.
