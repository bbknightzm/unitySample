How to use?

1. Set the identifier of your item which is set in App store, in the IAPSample script in the hierarchy of unity.
2. Build the project to iOS project, don't forget to set the bundle identifier.
3. Open the project in Xcode.
4. Add "StoreKit.framework" to the project.
5. Add "libInAppPurchase.a" to the project.
6. Add "InAppPurchase.h" file to the project.
7. Add "InAppPurchaseManager.mm" file to the project.
8. Then build it.
9. You'll see the app is running well in your device.
